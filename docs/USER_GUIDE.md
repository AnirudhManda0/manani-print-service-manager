# User Guide

## Dashboard

The dashboard shows:

- total prints today
- B&W pages
- color pages
- total services today
- total revenue today
- revenue trend for the last 7 days
- print vs service revenue contribution

## Print Log

The Print Log lists:

- time
- operator
- computer
- printer
- pages
- document
- print type
- paper size
- cost

Use the `Delete` button to remove a wrong transaction after confirmation.

## Services

Click any service button to record a service.
PrintX now shows a confirmation popup before the service is applied.

You can add services with arithmetic price expressions such as:

- `10 * 2`
- `5 + 5`
- `20 / 2`

## Reports

Reports support:

- daily view
- weekly view
- monthly view

Each report shows print totals, page totals, service totals, and revenue totals.

## Settings

Settings allow you to configure:

- server IP
- server port
- auto-discovery
- computer name
- operator ID
- auto-start
- polling interval
- print pricing
- retention mode
- backup folder

## Background Mode

If you close the main window, PrintX hides to the system tray instead of shutting down.
Monitoring continues in the background.
